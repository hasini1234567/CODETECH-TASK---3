Project Name: Movie Rating Analysis

1. Introduction

Data analysis helps understand audience preferences, movie performance, and rating trends. In this project, movie rating data is analyzed to identify highly rated movies, popular genres, audience voting patterns, and overall movie performance.

 2. Objective

The main objectives of this project are:

* Analyze overall movie ratings.
* Identify the highest-rated movies.
* Compare ratings across different genres.
* Study audience voting behavior.
* Analyze movie popularity based on votes.
* Visualize rating trends using charts and graphs.

3. Dataset Description

The dataset contains movie information with the following attributes:

* Movie Name
* Genre
* Rating
* Votes
* Release Year
* Duration
* Director
* Language
* Popularity Score
* Audience Reviews

4. Tools and Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook
* GitHub

5. Data Preprocessing

The following preprocessing steps were performed:

1. Created the movie dataset.
2. Checked for missing values.
3. Removed duplicate records.
4. Verified data types.
5. Cleaned and organized the data.
6. Prepared data for visualization and analysis.

6. Exploratory Data Analysis

6.1 Overall Rating Analysis

The average rating of all movies was calculated to understand overall movie performance.

Result:

* Average movie rating was calculated using the Rating column.

6.2 Genre-wise Rating Analysis

Ratings were compared among different movie genres.

Observation:

* Some genres received higher ratings than others.
* Genre analysis helps identify audience preferences.

6.3 Top Rated Movies Analysis

The highest-rated movies were identified.

Observation:

* Certain movies achieved exceptional ratings.
* These movies were considered audience favorites.

6.4 Votes Analysis

Audience voting patterns were analyzed.

Observation:

* Movies with higher votes generally indicated greater popularity.
* Vote count helped measure audience engagement.

6.5 Popularity Analysis

Movie popularity was studied using ratings and votes.

Observation:

* Highly rated movies often received a large number of votes.
* Popularity trends helped understand audience interests.

6.6 Rating Distribution Analysis

The distribution of movie ratings was analyzed.

Observation:

* Most movies received ratings within a specific range.
* Rating distribution provided insights into overall movie quality.

7. Data Visualization

The following visualizations were created:

1. Movie Rating Bar Chart
2. Genre-wise Average Rating Chart
3. Votes Distribution Pie Chart
4. Rating vs Votes Scatter Plot
5. Correlation Heatmap

These visualizations provide a better understanding of movie performance and audience behavior.

8. Results

The analysis revealed:

* Certain movies received significantly higher ratings.
* Some genres were more popular among audiences.
* Higher vote counts indicated greater audience engagement.
* Popular movies generally maintained strong ratings.
* Visualization helped identify rating trends and audience preferences.

9. Conclusion

The Movie Rating Analysis project successfully analyzed movie ratings and audience behavior. Through data cleaning, visualization, and statistical analysis, valuable insights were obtained regarding movie popularity, genre preferences, and audience engagement. These findings can help producers, streaming platforms, and analysts make informed decisions.
